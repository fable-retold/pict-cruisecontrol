# Pict-Cruisecontrol

> Automated and user-assisted web-page navigation and data marshaling.

- Workflow engine that drives a page through an ordered chain of steps
- Persistent workflow state that survives page reloads and navigation
- Extensible steps and assertions, plus an Electron page-injection example
